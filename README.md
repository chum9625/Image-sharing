# イメージ共有リポジトリ

制作で使うイメージを共有する。
## レスポンシブ Breakpoints

ChumTechで制作するものは[Bootstrap v5.0](https://getbootstrap.jp/docs/5.0/layout/breakpoints/) の基本に沿って設計する。

|Breakpoint|Class infix|Dimensions|
|---|:---:|:---:|
|X-Small|_None_|＜576px|
|Small|sm|≧576px|
|Medium|md|≧768px|
|Large|lg|≧992px|
|Extra Large|xl|≧1200px|
|Extra extra Large|xxl|≧1400px|
